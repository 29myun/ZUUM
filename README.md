# Screen Assist (Hackathon 2026)

Desktop app starter for the screen-share LLM assistant.

## Quick start

1. Install dependencies: `npm install`
2. Start dev mode: `npm run dev`

## What is included

- Electron main + preload bridge (IPC ping + source listing)
- Vite + React + TypeScript renderer
- UI scaffold for capture modes, preview, and snapshots

## Next milestones

- Region selection polish and multi-monitor support
- Floating glass overlay window
- LLM request pipeline and file context ingestion
